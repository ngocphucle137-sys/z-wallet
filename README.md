# Z-Wallet 🛡️
> The Next-Gen Privacy Wallet built for BNB Smart Chain (BSC).

Z-Wallet is an open-source, non-custodial wallet that utilizes zk-SNARKs technology to ensure your assets and transaction footprint remain 100% private. 

## ✨ Core Features
* **Absolute Self-Custody:** You hold the keys. No middlemen.
* **Zero-Knowledge Privacy:** Powered by advanced cryptography to break on-chain traceability.
* **BSC Native:** Lightning-fast transactions with ultra-low gas fees.
